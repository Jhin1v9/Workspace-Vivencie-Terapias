# Documentação do Projeto

Este diretório centraliza toda a documentação técnica e estratégica do projeto, organizada por tema.

## Estrutura

| Pasta | Conteúdo |
|-------|----------|
| `bug-detector/` | Documentação do sistema Bug Detector (comparativos, tutoriais, roadmaps, avaliações e uso local). |
| `bug-detector/images/` | Screenshots e imagens dos testes e fluxos do Bug Detector. |
| `auditoria/` | Relatórios de auditoria, checklists e notas de revisão do sistema. |
| `ai-setup/` | Configurações, prompts e payloads relacionados à integração com IA (Aura, DeepSeek, multi-provider). |
| `planejamento/` | Planos arquiteturais, correções aplicadas e relatórios de mudanças estruturais. |
| `misc/` | Documentos gerais e miscelânea (`AGENT_MEMORY.md`, `info.md`, etc.). |

## Arquivos mantidos na raiz

- `README.md` — Visão geral do projeto.
- `AGENTS.md` — Guia e convenções para agentes de código.
